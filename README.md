LifeLens AI is a premium, highly interactive Cognitive Operating System designed to enhance learning, productivity, and emotional well-being. Built with a stunning "vibrant cosmic" and Obsidian Glass aesthetic, the platform features a modern bento-box layout, deep glassmorphism, and fluid hardware-accelerated animations.

Under the hood, it harnesses the power of the Google Gemini API to drive intelligent insights, adversarial testing, and psychological regulation across 9 dynamic modules.

✨ Core Ecosystem & Features
🧠 High-Intensity Learning
The Feynman Simulator: Test your true understanding by breaking down complex topics into simple terms. The AI acts as a strict professor, grading your logic and generating visual alert cards for specific knowledge gaps.

The Hot Seat: A high-pressure, 60-second technical interview simulator wrapped in a stress-inducing terminal UI. Tests raw recall and execution under extreme time constraints.

Debate Arena: Engage in an adversarial "Devil's Advocate" chat. The AI actively tries to dismantle your argument while a dynamic, mathematical "Tug-of-War" progress bar tracks who holds the logical advantage.

AI Tutor Copilot: Your personalized conversational engine powered by Gemini. A context-aware assistant for live debugging, concept breakdowns, and interactive RAG-style learning.

⚡ Productivity & Visualization
Smart Dashboard: A dynamic command center featuring real-time data visualization via Chart.js, plotting your "Focus Trajectory" and tracking your overall "Session Mastery."

Neural Web (Knowledge Graphs): Visualize your expanding intellect. Bypassing standard lists, this module uses physics-based, force-directed 2D graphs to map your completed tasks as glowing, draggable constellations connected to a central brain.

Study Planner: An elegant objective sequencer featuring custom interactive states, neon-glowing checkboxes, and customized organizational frameworks.

🧘 Cognitive & Emotional Regulation
State Engine (Mood Regulation): An adaptive module that performs real-time sentiment analysis on your current emotional state. It dynamically unmounts the dashboard to generate immersive, full-screen stabilization exercises (like interactive particle physics or breathing loops) specifically tailored to calm your nervous system.

Flow State Engine: Keep your focus locked in with a custom Pomodoro workspace. Features massive minimalist typography and custom CSS-driven, synchronized "breathing" background orbs to sustain deep work.

🎨 Design Architecture
Premium Cosmic UI: An immersive frontend experience built with React, Tailwind CSS, and Framer Motion. Features a sophisticated dark cosmic color palette, multi-layered backdrop-blur glass panels, SVG noise overlays, and meticulously calculated flexbox transitions.

🛠 Tech Stack
Frontend: React.js, Vite, React Router DOM

Styling & Animations: Tailwind CSS, Framer Motion, Lucide React (Icons)

Data Visualization: Chart.js (react-chartjs-2), react-force-graph-2d

AI & Logic Engine: Google Gemini API

State Management: React Hooks (useState, useEffect, useRef), LocalStorage API
